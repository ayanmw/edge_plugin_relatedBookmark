# 关联书签查看器

一个基于 Edge 浏览器的扩展插件，用于快速查看当前页面的关联书签，支持相同主域名和 IP 的书签识别。

## 功能特性

- 📌 **关联书签识别**：自动识别与当前页面主域名或 IP 相同的书签
- 🎯 **当前页面高亮**：当前页面的书签会高亮显示，方便识别
- 📁 **完整路径显示**：显示每个书签的完整目录层级路径
- 🔗 **智能聚合功能**：一键将分散的关联书签聚合到同一目录（仅当需要时可用）
- 🎨 **简洁美观界面**：采用现代化设计，操作简单直观

## 安装方法

### 开发模式安装

1. 克隆或下载本仓库到本地
2. 打开 Edge 浏览器，访问 `edge://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展」
5. 选择本项目的根目录
6. 插件安装完成，可在浏览器右上角看到插件图标

## 使用说明

1. 打开任意网页
2. 点击浏览器右上角的插件图标
3. 插件会自动显示与当前页面相关联的所有书签
4. 查看关联书签的完整路径和详细信息
5. 当书签分散在不同目录时，可点击「一键全聚合」按钮将它们整理到同一目录

## 核心功能说明

### 关联书签识别

插件会根据当前页面的主域名和 IP 地址，查找并显示所有关联的书签。识别逻辑如下：

- 提取当前页面的主域名（如 `example.com`）
- 提取当前页面的 IP 地址
- 遍历所有书签，找出具有相同主域名或 IP 地址的书签
- 排除当前页面本身的书签（避免重复显示）

### 智能聚合功能

一键聚合功能会将分散在不同目录的关联书签，整理到一个新创建的目录中。聚合逻辑如下：

- 检查是否需要聚合（仅当书签数量 > 1 且不在同一目录时可用）
- 创建一个以当前主域名为名的新目录
- 将所有关联书签移动到这个新目录中
- 更新界面显示

## 技术栈

- **Manifest V3**：采用最新的 Chrome 扩展开发规范
- **JavaScript**：核心开发语言
- **HTML/CSS**：界面设计
- **Chrome API**：使用 Bookmarks API 和 Tabs API
