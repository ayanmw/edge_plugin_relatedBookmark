# 关联书签查看器

一个强大的 Edge 浏览器扩展，帮助您快速管理和聚合书签。

## 功能特性

### 🎯 关联书签查看
- 自动识别当前页面的关联书签
- 基于相同主域名或 IP 地址的智能匹配
- 高亮显示当前页面的书签

### 🔍 强大的搜索功能
- 支持多种搜索范围：
  - **标题** - 搜索书签标题
  - **域名** - 搜索 URL 中的域名部分
  - **URL 查询** - 搜索 URL 中的查询参数
  - **目录名** - 搜索书签目录，并包含该目录下的所有书签
- 支持大小写敏感/不敏感搜索
- 一键全选/反选搜索选项
- 实时显示搜索结果

### � 书签聚合
- 一键将多个书签聚合到同一目录
- 支持自定义目标目录
- 支持创建新的聚合目录
- 智能判断：当书签数量 ≤ 1 或所有书签在同一目录时自动禁用聚合按钮

### 🗑️ 书签管理
- 直接删除不需要的书签
- 搜索结果中可移除书签（不删除实际书签，仅从搜索结果中过滤）
- 显示书签完整路径信息

## 安装方法

### 开发者模式安装

1. 下载或克隆此项目到本地
2. 打开 Edge 浏览器
3. 访问 `edge://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择项目文件夹

## 使用方法

### 查看关联书签

1. 访问任意网页
2. 点击浏览器工具栏中的扩展图标
3. 自动显示当前页面的关联书签列表
4. 点击"一键全聚合"将所有关联书签移动到同一目录

### 搜索书签

1. 点击右上角的搜索按钮（🔍）
2. 输入搜索关键词
3. 勾选需要的搜索范围：
   - 标题：搜索书签标题
   - 域名：搜索 URL 域名（如 github.com）
   - URL 查询：搜索 URL 参数（如 ?id=123）
   - 目录名：搜索目录并包含该目录下所有书签
4. （可选）勾选"区分大小写"进行大小写敏感搜索
5. 点击"搜索"按钮或按回车键
6. 使用"移除"按钮筛选要聚合的书签
7. 点击"一键聚合"完成聚合

### 聚合书签

1. 在关联书签列表或搜索结果中
2. 点击"一键全聚合"按钮
3. 选择聚合方式：
   - **新建聚合目录**：创建新目录并移动书签
   - **自定义目录**：选择现有目录作为目标
4. 点击"完成"执行聚合
